# algo_api

Библиотека для бекенда Алгоритмики для учеников.
> Данная библиотека не имеет отношения к Алгоритмике

## Установка

Клонируйте репозиторий и установите с помощью `pip`

```bash
git clone https://github.com/moontr3/algo_api.git
cd algo_api
pip install *
```
